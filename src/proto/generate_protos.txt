1. Navigate to parent file directory level:
.../lnd_grpc/src/

2. Make sure googleapis is cloned in the same folder:
git clone https://github.com/googleapis/googleapis.git

3. Make sure you are running in venv

4. Run command to generate proto gRPC files:
python -m grpc_tools.protoc --proto_path=googleapis:. --python_out=. --grpc_python_out=. lnd_grpc/proto/rpc.proto
